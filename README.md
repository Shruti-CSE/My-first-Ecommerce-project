# My-first-Ecommerce-project
Data Analytics Project: Customer Segmentation and Revenue Analysis."

This project focuses on transforming raw transactional data into actionable business insights. Using a combination of SQL for data engineering and Python for data visualization, I analyzed sales trends and performed customer segmentation to identify high-value users.

🛠️Tech Stack
Language: Python 3.x
Database: SQLite
Libraries: Pandas, Matplotlib, Sqlite3, IPython-SQL (SQL Magic)

🚀 Key Features
Database Integration: Automated the conversion of raw CSV/JSON-style data into a structured SQL database (mtech_project.db).
Customer Segmentation: Implemented a Tiering System (Platinum, Gold, Silver) using SQL CTEs (Common Table Expressions) and Window Functions to rank customers based on Lifetime Value (LTV).
Growth Analysis: Visualized monthly revenue trends, identifying a 75% increase in revenue from January ($1,650) to March ($2,900).
Advanced SQL: Utilized RANK(), CASE statements, and strftime for time-series analysis.

📊 Visualizations
The project includes:
Revenue Growth Trend: A line chart showing the month-on-month trajectory.
Customer Tier Distribution: A categorical analysis identifying the concentration of premium vs. standard customers.
